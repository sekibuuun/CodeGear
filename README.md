# CodeGear

## 概要

エンジニアのためのQRコード生成アプリ

## 使い方

1. SNSを選ぶ
2. ユーザーIDを入力
3. QRコードを生成
4. 保存もできる

## 使用技術

- 大枠
  - Vite
  - React
  - TypeScript
- QRコード生成
  - qr-code-styling-2
- UI
  - shadcn/ui
  - Tailwind CSS
- デプロイ
  - GitHub Pages
- コードレビュー
  - CodeRabbit
